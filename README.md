# 🤖 AI Resume Scanner (ATS Checker)

एक बेहतरीन और आधुनिक **AI-powered Resume Scanner (ATS Checker)** जो आपके रिज्यूमे को किसी भी जॉब डिस्क्रिप्शन (Job Description - JD) के अनुसार स्कैन और ऑप्टिमाइज़ करता है। यह प्रोजेक्ट **Streamlit** और Google के लेटेस्ट **Gemini 2.5 Flash** मॉडल का उपयोग करके बनाया गया है।

---

### ✨ Features (विशेषताएं)
* **Match Percentage:** रिज्यूमे और जॉब डिस्क्रिप्शन के बीच का मैच प्रतिशत दिखाता है।
* **Missing Keywords:** उन मुख्य स्किल्स और कीवर्ड्स को पहचानता है जो आपके रिज्यूमे से गायब हैं।
* **Profile Recommendations:** रिज्यूमे को बेहतर बनाने के लिए AI द्वारा कस्टमाइज़्ड सुझाव देता है।
* **PDF Support:** सीधे PDF फाइल अपलोड करने की सुविधा।

---

## 🛠️ Tech Stack (तकनीकी उपकरण)
* **Frontend/UI:** Streamlit
* **AI Model:** Google Gemini 2.5 Flash API
* **Language:** Python 3.14
* **PDF Parser:** PyPDF2
